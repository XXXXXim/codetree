💊 처방전 3: 코드트리(CodeTree) 활용법 (기초가 흔들릴 때)
코드트리 레슨 6면 아마 "다중 반복문(이중 for문)" 파트일 거야. 여기가 초보자들이 제일 많이 좌절하는 1차 통곡의 벽이야. 여기서 막히는 건 너무 당연해.

진도 욕심 버려: 레슨 6가 막히면 레슨 4(단순 반복문)나 5(조건문)가 100% 숙달 안 된 거야. 부끄러워하지 말고 앞으로 돌아가서 쉬운 문제를 안 보고 3분 안에 짤 수 있을 때까지 반복해.

그림 그리기: 이중 for문(i, j)이 돌 때 (0,0), (0,1), (0,2)... 이렇게 변하는 과정을 종이에 표를 그려서 손으로 따라가 봐. 눈으로만 보면 절대 안 늘어.
